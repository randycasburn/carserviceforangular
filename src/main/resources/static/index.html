Welcome to the Car Service

  • Get All Cars (/cars)
  • Sort cars by price: (/cars?filter=price)
  • Sort cars by model: (/cars?filter=model)
  • Sort cars by make: (/cars?filter=make)
  • Sort cars by number of doors: (/cars?filter=doors)
  • Sort cars by year: (/cars?filter=year)
Optionally specify number of cars to return ( &count=3 ) with any query above
  • Get first three Cars sorted by price (/cars?filter=price&count=3)
